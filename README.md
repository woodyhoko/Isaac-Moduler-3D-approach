# Isaac Modular 3D Approach

A research and experimentation repository exploring **modular 3D scene composition** using NVIDIA Isaac Sim — investigating how to structure reusable 3D components for robotics simulation environments.

---

## Context

[NVIDIA Isaac Sim](https://developer.nvidia.com/isaac-sim) is a robotics simulation platform built on Omniverse. This project explores modular approaches to building 3D simulation environments — making it easier to compose and reuse scene elements across different robotics tasks.

---

## Topics

- Modular asset composition in USD (Universal Scene Description)
- Scene parameterization for domain randomization
- Reusable environment building blocks for RL training

---

## Requirements

- NVIDIA Isaac Sim (Omniverse)
- Python 3.8+
