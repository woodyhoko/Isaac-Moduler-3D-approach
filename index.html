NVIDIA Isaac Sim · Omniverse · USD

Compose worlds,
don't hand-build them.

Training robot RL policies needs many diverse simulation environments — and hand-authoring each one is slow and yields brittle policies. This repo builds a library of parametric USD blocks — rooms, furniture, lighting, robots — that assemble and randomize into a huge training distribution.

📖 Read the Approach ⌥ Source 🤖 Isaac Sim
why USD
Reference-based, not embed-based.
🧱

Modular prims

Each component is a USD prim under an Xform; a single asset file is referenced across every scene that uses it.

🔀

Variant sets

Swap room layouts, furniture configs and robots at episode reset via the Python API — no rebuild needed.

🎲

Domain randomization

Geometry, materials, lighting, physics and sensor noise are sampled each reset to close the sim-to-real gap.

📷

Annotator sensors

RGB, depth and normals annotators on repositionable camera rigs generate synthetic training data.
